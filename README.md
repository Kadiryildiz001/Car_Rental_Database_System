# Car_Rental_Database_System
Araac_VT | MS SQL Database Project  Bu proje, bir araç yönetim sisteminin ilişkisel veritabanı altyapısını içerir. T-SQL ile geliştirilen sistemde; normalizasyon kurallarına uygun tablo yapıları, veri bütünlüğü ve ilişkisel modelleme (ER) uygulanmıştır.  Teknolojiler: MS SQL Server, T-SQL Kurulum: .sql scriptini SSMS üzerinden çalıştırın.
